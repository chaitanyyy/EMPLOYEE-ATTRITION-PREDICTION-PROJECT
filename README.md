PROJECT SUMMARY
===============
This project uses Machine Learning to predict employee attrition and identify 
at-risk employees, helping companies reduce turnover costs.

KEY RESULTS
===========
- Model Accuracy: 87.07%
- ROC-AUC Score: 0.912
- High-Risk Employees Identified: 294 out of 1,470
- Top Attrition Driver: Overtime work (2.9x higher risk)
- Potential Annual Savings: $3.6 million

TECHNOLOGIES USED
=================
- Python 3.13
- Libraries: Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn
- Machine Learning Models: Logistic Regression, Random Forest
